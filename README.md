## :rocket: Instalação

:bulb: Para instalar os pacotes, você precisará ter o [Composer](https://getcomposer.org/) instalado em sua máquina. Após ter instalado, entre na pasta `src` usando o Terminal/CMD e digite o seguinte comando:

```
composer update
```
